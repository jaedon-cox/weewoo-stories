Privacy Policy

Last updated: April 2026

AutoStory is a personal automation tool that uses the TikTok API solely to upload AI-generated videos to a single owned TikTok account. It does not collect, store, or share any personal data from other users.

What the app accesses: The app requests permission to upload videos to the authenticated account. No data is read from followers, comments, messages, or any other account.

Data storage: No user data is transmitted to or stored on any third-party server beyond what is required by the TikTok API itself.

Third parties: No user data is sold, rented, or shared with any third party.
